# danielensign.com

Personal landing page with social links and project portfolio.

## Projects

- **[Golf Squares](https://golfsquares.com/)** — Wikipedia-style view of golfer performance in majors since 1995
- **[Grocery List](https://github.com/densign01/grocery-list-skill)** — Claude Code skill that adds recipes and groceries to Apple Reminders
- **[Speed Reader](https://speed-reader-2gtu.vercel.app/)** — Uses RSVP technique to help you read faster
- **[Garmin Workout Sync](https://garmin-sync.vercel.app/)** — Tool for syncing workouts in plain text to Garmin watches

## Development

Static HTML site with no build system. Open `index.html` in a browser to preview.

## Deployment

Hosted on GitHub Pages. Push to `main` to deploy.
